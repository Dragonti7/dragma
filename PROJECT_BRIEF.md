# Project Title: AI Fortune Cookie (AI 포춘 쿠키)

## 1. 프로젝트 개요
사용자가 화면의 포춘 쿠키를 클릭하면, 쿠키가 깨지는 애니메이션과 함께 매번 새로운 운세 메시지를 보여주는 간단한 웹 애플리케이션입니다. 
Gemini API를 활용하여 매번 색다르고 재치 있는 운세를 생성합니다.

## 2. 기술 스택 (Tech Stack)
- **Frontend:** HTML5, CSS3, Vanilla JavaScript (복잡한 프레임워크 없이 가볍게 구축)
- **AI Integration:** Google Gemini API (텍스트 생성용)
- **Deploy:** Vercel 또는 GitHub Pages (추후 배포 예정)

## 3. 주요 기능 (Features)
### A. 메인 화면
- 화면 중앙에 먹음직스러운 '포춘 쿠키' 일러스트 배치 (CSS 또는 SVG 활용).
- "당신의 오늘 운세를 확인하세요"라는 문구 표시.

### B. 인터랙션 (Interaction)
- 사용자가 쿠키를 클릭하면:
  1. '바사삭' 갈라지는 시각적 애니메이션 (CSS Animation) 실행.
  2. (선택 사항) 가벼운 효과음 재생.
  3. 로딩 인디케이터 표시 (API 응답 대기 중).

### C. 운세 생성 (AI Generation)
- Gemini API에 프롬프트를 보내 운세를 받아옴.
- **프롬프트 예시:** "재치 있고 긍정적인 한 문장 운세를 한국어로 만들어줘. 이모지 1개를 포함해줘."
- 결과 텍스트를 쿠키가 갈라진 틈이나 하단에 카드 형태로 우아하게 표시.

### D. 다시 하기
- "다른 운세 보기" 버튼을 누르면 쿠키가 다시 합쳐지거나 새로운 쿠키가 나타나는 리셋 기능.

## 4. 디자인 가이드 (UI/UX)
- **테마:** 파스텔 톤 배경 (연한 베이지 or 핑크), 편안하고 신비로운 느낌.
- **폰트:** 가독성 좋은 한글 웹 폰트 (예: Pretendard, Noto Sans KR).
- **애니메이션:** 끊김 없이 부드러운 모션 (Smooth Easing).

## 5. 구현 단계 (Step-by-Step Implementation)
안티그래비티 에이전트는 아래 순서대로 작업을 진행하고, 각 단계마다 브라우저 테스트를 수행해주세요.

1. **Step 1: 기본 구조 잡기**
   - `index.html`, `style.css`, `script.js` 파일 생성.
   - 중앙 정렬된 기본 레이아웃과 쿠키 이미지(Placeholder) 배치.

2. **Step 2: 애니메이션 구현**
   - CSS를 사용하여 쿠키가 클릭 시 좌우로 갈라지는 애니메이션 구현.

3. **Step 3: 가짜 데이터(Mock Data) 연동**
   - API 연결 전, 자바스크립트 배열에 저장된 문구 5개를 랜덤으로 출력하는 기능 구현.
   - UI 동작 확인.

4. **Step 4: Gemini API 연동**
   - 실제 API 호출 함수 작성 (API Key는 환경 변수 처리).
   - 로딩 상태 및 에러 처리(네트워크 오류 시 기본 문구 출력).

5. **Step 5: 폴리싱 (Polishing)**
   - 폰트 적용, 모바일 반응형 확인, 디자인 다듬기.

- API Key가 없을 경우를 대비해 데모 모드(Mock Data)가 우선 작동하도록 할 것.
